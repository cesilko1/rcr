# React call rest

React call rest - typesafe react hooks for simple calling rest api.

This package already contains typescript typings.

## Installation

### npm
```sh
npm i @vilem/rcr
```

### yarn
```sh
yarn add @vilem/rcr
```

Available hooks:

  * `useFetchApi`
  * `useLazyFetchApi`
  * `useMutateApi`
